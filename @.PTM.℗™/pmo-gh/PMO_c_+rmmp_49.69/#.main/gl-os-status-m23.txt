## Reference Architecture (from Python TPM_OS)

**What GL-OS Should Be (based on #.ref/TPM_OS.py+3.9]ON!EST_f13/):**

```
┌─────────────────────────────────────────────────────────────┐
│              GL-OS Window (OpenGL/Qt Desktop)               │
│  BG: #000044 (dark blue)                                    │
│                                                             │
│  ┌─────────────────┐  ┌─────────────────┐                  │
│  │  Terminal App   │  │  File Browser   │  ← Widgets/Apps  │
│  │  (Qt widget)    │  │  (Qt widget)    │    inside GL     │
│  └─────────────────┘  └─────────────────┘                  │
│                                                             │
│  Right-click → Context Menu (Qt QMenu)                     │
│    [New Terminal] [New Folder] [New File]                  │
│                                                             │
│  Windows: Draggable, resizable (MiniWindow class)          │
└─────────────────────────────────────────────────────────────┘
```

**Key Components from Python Reference:**

1. **Desktop Plugin** (`plugins/desktop/desktop.py`):
   - Creates Qt QApplication and MainWindow
   - Manages DesktopWidget as central widget
   - Handles right-click → logs to master_ledger

2. **Terminal Plugin** (`plugins/desktop/terminal.py`):
   - Watches master_ledger for `EventFire: open_terminal`
   - Creates TerminalApp widget inside MiniWindow
   - Adds window to DesktopWidget

3. **Window Manager** (`window_manager/mini_window.py`):
   - Draggable, resizable windows
   - Title bar, close button
   - Parent = DesktopWidget

4. **Engine** (`engine.py`):
   - PluginManager loads all plugins
   - MasterLedgerMonitor watches for events
   - Qt event loop runs everything

## TPM-Compliant C Implementation

**Challenge:** Convert Qt-based desktop to C with OpenGL (gl_renderer.c pattern)

**Architecture:**

```
pieces/apps/gl_os/
├── plugins/+x/
│   ├── gl_desktop.+x      # Main OpenGL window (like desktop.py)
│   ├── gl_terminal.+x     # Terminal widget (like terminal.py)
│   └── gl_window.+x       # Window manager (like mini_window.py)
├── session/
│   ├── state.txt          # Desktop state (windows, positions)
│   ├── history.txt        # Input history (mouse/keyboard)
│   └── view.txt           # Rendered desktop view
└── pieces/
    ├── desktop/           # Desktop piece
    └── terminal/          # Terminal piece
```

**gl_desktop.c should:**
1. Open OpenGL window (800x600, bg #000044)
2. Use GLUT/OpenGL like gl_renderer.c
3. Manage window list (positions, titles, content)
4. Handle mouse (drag, click, right-click menu)
5. Watch master_ledger for events
6. Spawn gl_terminal.+x as child window

**TPM Compliance:**
- All state in `pieces/apps/gl_os/session/`
- Events logged to master_ledger
- Pieces own their state (desktop/, terminal/)
- Separate from CHTPM terminal session
