-------------------------------

TECHNICAL SPECIFICATION:

1. Terminal Command Recognition:
   - Modify apps/terminal_app.py to recognize fuzzpet commands
   - Commands should be top priority in execute_command()
   - Format: [timestamp] MethodCall: {command} on fuzzpet | Caller: terminal

2. Game Processing:
   - Game plugin (plugins/core/game.py) monitors master ledger
   - Processes MethodCall events targeting fuzzpet piece
   - Updates state according to command rules:
     * feed: decreases hunger, triggers 'fed' event
     * play: increases happiness, triggers 'played' event  
     * sleep: increases energy, triggers 'slept' event
     * status: displays current stats
     * evolve: increases level, triggers 'evolve' event

3. Response Mechanism:
