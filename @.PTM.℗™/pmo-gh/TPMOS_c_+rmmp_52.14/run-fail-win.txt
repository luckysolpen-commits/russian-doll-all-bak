## ROOT CAUSE FOUND AND FIXED ✓

### The Problem: Path Format Mismatch

**C code on Windows** needs Windows-style paths:
```
project_root=C:\Users\jbro8\...\TPMOS_c_+rmmp_52.01
```

**Shell scripts** were writing bash-style paths:
```
project_root=/c/Users/jbro8/.../TPMOS_c_+rmmp_52.01
```

When gl_desktop.c tried to `fopen("/c/.../frame_history.txt")`, Windows couldn't understand `/c/...` format!

### The Fix: Use cygpath

Shell scripts now use `cygpath -w` to convert bash paths to Windows paths:

```bash
WIN_PATH=$(cygpath -w "$(pwd)")
# WIN_PATH = "C:\Users\jbro8\...\TPMOS_c_+rmmp_52.01"

cat > pieces/locations/location_kvp << EOF
project_root=${WIN_PATH}
EOF
```
