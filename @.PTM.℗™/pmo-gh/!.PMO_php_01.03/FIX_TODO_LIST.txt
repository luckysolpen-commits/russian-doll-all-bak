## Key Nuances from C Version

### Navigation Mode (active_index == -1)
- Arrow keys (w/s/a/d or 1000-1003) move focus between interactive elements
- Focus indicator: [>] for focused, [ ] for unfocused
- Number keys jump to element by interactive index
- Enter (13) activates: follows href, opens menu, or triggers onClick
- Nav buffer shows last key pressed (w/s/a/d)

### INTERACT Mode (active_index points to INTERACT element)
- All keys are injected into fuzzpet/history.txt
- ESC (27) exits INTERACT mode
- Nav buffer shows last key or "[Key]"

### Menu Behavior
- <menu> tag itself is interactive (counted)
- Children are NOT counted as interactive until menu is opened
- Opening menu: sets active_index to menu, enables child navigation
- Child navigation: uses current_child_focus index

### Variable Loading (load_vars equivalent)
1. Load defaults from .pdl files
2. Override with runtime state.txt (Mirror)
3. Load clock state
4. Load display state
5. Load module's view.txt as ${game_map}

### Map Rendering Format
- Each line padded to 60 chars with "║" borders
- Map uses: # (wall), . (floor), @ (pet), T (tree), R (rock)
- Response text shown above map
- DEBUG line shows last key code
