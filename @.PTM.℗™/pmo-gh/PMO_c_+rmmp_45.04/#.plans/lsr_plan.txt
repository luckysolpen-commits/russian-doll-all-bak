LSR (LUNAR STREETRACE RAIDER) MASTER SPECIFICATION
====================================================
**Prompt/Directive for AI Implementation**

1. PROJECT OVERVIEW
-------------------
**Title:** LSR (Lunar Streetrace Raider)
**Concept:** A "Civ-Lite" + "Life Sim" + "Market Economy" hybrid where autonomous AI agents (FuzzPets) work, reproduce, research, and build civilizations on a lunar map.
**Core Differentiator:** R&D is driven by local LLMs (via `ai-labs`), generating actual recipes/techs rather than a static tree.
**Architecture:** TPM-Compliant Project (`projects/lsr/`).

2. PHASE 1: THE FOUNDATION (MSR-LITE)
-------------------------------------
**Objective:** Clone the core economic loop of MSR but scaled down.

- **Scope:** 
    - 4 Companies (e.g., LunarMining, MoonFarms, CraterChem, HelixLabs).
    - 1 Government (The Lunar Authority).
    - Basic Market: Commodities (Ore, Food, Chems, Tech).
- **Mechanics:**
    - Companies hire FuzzPets.
    - Companies produce goods based on labor + resources.
    - Government taxes transactions and enforces "Law" (basic boolean flags for now).

3. PHASE 2: THE BIOLOGICAL ENGINE (FADAM & FEVE)
------------------------------------------------
**Objective:** Implement the Life Sim layer (The Sims/FuzzPet logic).

- **Progenitors:** Start with 2 entities: 'Fadam' (Male) and 'Feve' (Female).
- **Stats:**
    - `hunger`, `energy`, `bladder`, `social`, `health`.
    - `genetics`: Traits passed to offspring (e.g., +Intelligence, +Strength).
- **Loop:** 
    - Agents must buy food (economy interaction).
    - Agents must sleep (housing/map interaction).
    - Agents must work (to earn money).
    - Reproduction: Requires high `social` + `energy`. Creates new unique Agent piece.

4. PHASE 3: THE MAP & CIV MECHANICS
-----------------------------------
**Objective:** 2D/3D Z-Layered Tile Map (Civilization Style).

- **Map Structure:** `pieces/world/map_lunar/`.
- **Territory:**
    - **Government Land:** Safe, taxed, law enforcement present.
    - **Company Claims:** Land owned by corps.
    - **Unclaimed/Wild:** "Barbarian" zones.
- **Building:**
    - Agents can build structures (Houses, Factories, Labs).
    - **Criminal Mechanic:** If a Company/Structure is built on unclaimed land without Gov permit, it is "Criminal". It pays no tax but risks being "Raided" by Law Enforcement.

5. PHASE 4: AI-DRIVEN R&D (THE GEMMA LINK)
------------------------------------------
**Objective:** Use `ai-labs` ops to generate content dynamically.

- **Tech Tree:** Not static.
- **Workflow:**
    1. Agent works at `HelixLabs` on "R&D".
    2. LSR Manager calls `ai-labs` op: `generate_tech "Chemistry" "Lunar Soil"`.
    3. LLM (Ollama/Gemma) returns a JSON recipe: `{ "name": "Mooncrete", "inputs": ["Regolith", "Water"], "strength": 50 }`.
    4. This new item is added to the game's item database (`projects/lsr/pieces/registry/items.txt`).
- **Scope:** Drugs, Tools, Vehicles, Recipes, Physics constants.

6. PHASE 5: P2P MULTIPLAYER
---------------------------
**Objective:** Allow interaction between different LSR instances.

- **Protocol:** Uses `p2p-net` ops.
- **Interaction:**
    - Visiting: Send Agent piece to another player's IP (Travel).
    - Trade: Sell "Mooncrete" formula to another player.
