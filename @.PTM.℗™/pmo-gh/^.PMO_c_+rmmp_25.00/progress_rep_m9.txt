1. THE VISION: EDITOR vs PLAYER (THEY ARE DIFFERENT!)
================================================================================

### EDITOR (Map/Event Creation Tool)
```
╔══════════════ P I E C E M A R K   E D I T O R ════════════╗
║  [>] 1. [[MAP]] [ ] 2. [[PIECES]] [ ] 3. [[DB]] [ ] 4. [[EVENTS]]      ║
╠═══════════════════════════════════════════════════════════╣
║  ####################             │  PALETTE:              ║
║  #  ...............T#             │  [1] # Wall            ║
║  #  ..@.........Z..T#             │  [2] . Floor           ║
║  #  ....R..........T#             │  [3] R Rock            ║
║  #  ....R..........T#             │  [4] T Tree            ║
║  #  ..X.R..?.......T#             │  [5] ? Mystery         ║
║  #  ....R..........T#             │                        ║
║  #  ................#             │  TOOL: [Paint]         ║
║  #  #################             │  POS: (5, 3, 0)        ║
╠═══════════════════════════════════════════════════════════╣
║ [RESP]: Tile placed.                                      ║
║ [Z-LEVEL]: 0   | [ACTIVE]: selector   | [KEY]: SPACE      ║
╠═══════════════════════════════════════════════════════════╣

╚═══════════════════════════════════════════════════════════╝
[1-4]: Switch Tab | [WASD]: Move | [SPACE]: Use Tool | [Q]: Quit
```

**Purpose:** Create maps, place pieces, write event scripts
**Key Features:**
- 4 Tabs: MAP, PIECES, DB, EVENTS
- Cursor-based navigation (not player character)
- Tools: Paint, Pick, Fill, Rectangle
- Palette: Select tiles to place
- Saves to projects/{project_id}/maps/ and pieces/

================================================================================
2. KEY DIFFERENCES
================================================================================

| Aspect          | EDITOR                          | PLAYER                        |
|-----------------|---------------------------------|-------------------------------|
| **Cursor**      | Selector cursor (X)             | Player character (@)          |
| **WASD**        | Moves cursor                    | Moves player character        |
| **SPACE**       | Use tool (paint/pick)           | Interact with entity          |
| **1-4**         | Switch tabs                     | Call piece methods            |
| **Output**      | Writes to maps/, pieces/        | Reads from maps/, pieces/     |
| **Scripts**     | Creates .asm scripts            | Executes .asm scripts         |
| **Goal**        | Create content                  | Experience content            |

================================================================================
3. SHARED OPS (Both Use Same Library)
================================================================================

Both Editor and Player call the SAME Ops:
- `move_player.+x` - Editor moves cursor, Player moves character
- `move_z.+x` - Both change Z-levels
- `place_tile.+x` - Editor places tiles (Player doesn't use this)
- `interact.+x` - Player interacts with entities (Editor doesn't use this)
- `render_map.+x` - Both render the map view
- `console_print.+x` - Both show messages
- `layout_op.+x` - Both show menus

**The Op doesn't know if it's being called from Editor or Player.**
It just does its job and updates state.txt.
