## KEY INSIGHT: WHY COPYING C WORKS

The C implementation is the **reference architecture**. Every time we deviated (state machine, regex substitution, os.system()), we introduced bugs. Every time we copied C exactly (tokenizer, render_element, process_key), it worked.

**The Pattern:**
1. Read C code
2. Understand the algorithm
3. Translate line-by-line to Python
4. Test against C output
