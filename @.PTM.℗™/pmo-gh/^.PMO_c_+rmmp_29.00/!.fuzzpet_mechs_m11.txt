# FUZZPET MECHANICS MAPPING - M11
Goal: Achieve 100% parity with legacy Fuzzpet using Op-Driven Architecture.

## 1. CORE STATS (THE SOUL)
| Mechanic | Legacy Implementation | Player V2 Op | Status |
|----------|-----------------------|---------------|--------|
| Hunger   | `hunger++` on move    | `stat_decay`  | ✅ Basic |
| Energy   | `energy--` on move    | `move_entity` | ✅ Basic |
| Happy    | `happiness` from 'T'  | `move_entity` | ✅ Basic |
| Level    | `level++` manual      | `fuzzpet_action`| ✅ Basic |

## 2. INTERACTION (THE MUSCLES)
| Action | Legacy Key | V2 Op | Result |
|--------|------------|-------|--------|
| Feed   | Key 2      | `fuzzpet_action fuzzpet feed` | Hunger -20 |
| Play   | Key 3      | `fuzzpet_action fuzzpet play` | Happiness +10 |
| Sleep  | Key 4      | `fuzzpet_action fuzzpet sleep`| Energy +30 |
| Evolve | Key 5      | `fuzzpet_action fuzzpet evolve`| Level +1 |
