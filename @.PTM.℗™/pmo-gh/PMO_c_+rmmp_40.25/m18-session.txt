## Modular Artifact Contract (V1)
The `project.pdl` file in `projects/<project_id>/` is the canonical manifest.

### 1. META Section
- `project_id`: Canonical name (must match directory name).
- `entry_layout`: (New) Path to the CHTPM layout used to "boot" this project.
- `version`: SemVer compatible versioning.

### 2. STATE Section
- `starting_map`: Map file to load on boot (e.g., `map_01`).
- `player_piece`: Piece ID representing the user's primary entity.
- `current_world`: (New) Active world ID (default `world_01`).
- `current_map`: (New) Active map ID (default `map_01`).

### 3. Pieces & World Structure
- Canonical: `projects/<project>/pieces/<world_id>/<map_id>/<piece_id>/`
- Fallback: `projects/<project>/pieces/<piece_id>/`
