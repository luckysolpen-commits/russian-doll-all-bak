# 16.fuzz-editor.txt - RMMP Editor Modernization Plan
# Status: ✅ COMPLETED
# Goal: Adopt FuzzPet selector movement and dynamic interaction patterns.

## 1. THE VISION
The RMMP Editor (Option 1 in Project Manager) has been transitioned to the "Sovereign Trait" pattern. It now uses standardized ops and dynamic interaction tags, making it the primary model for project-aware applications.

## 2. PHASE 1: LAYOUT UPDATE (editor.chtpm) - ✅ DONE
- Updated `pieces/apps/editor/layouts/editor.chtpm`.
- Added `<interact src="pieces/apps/player_app/history.txt" />`.
- Result: Editor now shares the unified interaction buffer, allowing for seamless project swapping.

## 3. PHASE 2: MODULE REFACTOR (editor_module.c) - ✅ DONE
- **Input Polling:** Simplified to unconditionally poll `pieces/apps/player_app/history.txt`.
- **Selector Movement:** 
    - Replaced hardcoded logic with shared `move_entity.+x` trait.
    - Call: `system("./pieces/apps/playrm/ops/+x/move_entity.+x selector w template")`
    - Refactored to pass `current_project` explicitly, ensuring correct project pieces are updated.
- **State Sync:** Module reloads `cursor_x/y` from `projects/{id}/pieces/selector/state.txt` after each move.
- **Compile Clean:** Implemented `asprintf` across all paths to silence truncation warnings and meet engineering standards.

## 4. PHASE 3: PARSER ENHANCEMENT (chtpm_parser.c) - ✅ DONE
- **Dynamic SRC:** Enhanced `<interact>` tag handler.
- **Substitution:** Added `substitute_vars()` to the `src` attribute.
- **Impact:** Layouts can now use dynamic paths like `<interact src="projects/${project_id}/history.txt" />`.

## 5. PHASE 4: SELECTOR PIECE STANDARDIZATION - ✅ DONE
- Ensured `template` and other projects have standardized `pieces/selector/` piece.
- `state.txt` now includes `type=selector` and `on_map=1` for reliable rendering.

## 6. VERIFICATION - ✅ VERIFIED
- "Control Map" enters INTERACT mode.
- Arrows/WASD move 'X' correctly.
- Boundaries (1-18, 1-8) enforced by `move_entity.+x`.
- Selection (ENTER) works: correctly scans for pieces under selector.

"The RMMP Editor is now the gold standard for TPM-compliant tools."
