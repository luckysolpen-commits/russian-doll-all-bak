## 2. EDITOR_MODULE.C UPDATES

### 2.1 Selector Movement (Op-Driven)

**BEFORE (Hardcoded):**
```c
if (key == 'w' || key == 'W' || key == 1002) cursor_y--;
else if (key == 's' || key == 'S' || key == 1003) cursor_y++;
else if (key == 'a' || key == 'A' || key == 1000) cursor_x--;
else if (key == 'd' || key == 'D' || key == 1001) cursor_x++;
```

**AFTER (Op-Driven):**
```c
if (key == 'w' || key == 'W' || key == 1002) {
    system("./pieces/apps/playrm/ops/+x/move_selector.+x up > /dev/null 2>&1");
}
else if (key == 's' || key == 'S' || key == 1003) {
    system("./pieces/apps/playrm/ops/+x/move_selector.+x down > /dev/null 2>&1");
}
// ... sync cursor_x, cursor_y from selector/state.txt
```

**Benefits:**
- Selector state persisted in `pieces/apps/editor/pieces/selector/state.txt`
- Movement logic is reusable (fuzzpet, player, editor all use same op)
- Frame marker hit automatically by op

### 2.2 Ctrl+Z Undo Support

**NEW:**
```c
// Ctrl+Z Undo
if (key == 122) {  // Ctrl+Z ASCII
    char cmd[MAX_PATH];
    snprintf(cmd, sizeof(cmd), "%s/pieces/apps/playrm/ops/+x/undo_action.+x > /dev/null 2>&1", project_root);
    system(cmd);
    set_response("Undo performed");
}
```

**Ledger Format:**
```
PLACE_TILE|map_0001|5|5|.|#
CREATE_PIECE|npc|7|3
DELETE_PIECE|chest|10|8
```

**Undo Logic:**
- `PLACE_TILE` → Reverts to old tile
- `CREATE_PIECE` → Deletes created piece
- `DELETE_PIECE` → Cannot undo (no backup)

### 2.3 Entity Creation on Glyph Paint

**BEFORE:**
```c
// Only placed terrain
system("./place_tile.+x map 5 5 @");
```

**AFTER:**
```c
const char *glyph = active_set[glyph_idx];
int is_entity = (strcmp(glyph, "@") == 0 || strcmp(glyph, "&") == 0 || 
                 strcmp(glyph, "Z") == 0 || strcmp(glyph, "T") == 0);

if (is_entity) {
    // Create piece
    system("./create_piece.+x @ 5 5");
    set_response("Piece created");
} else {
    // Place terrain
    system("./place_tile.+x map 5 5 #");
}
```

**Entity Type Mapping:**
| Glyph | Piece Type | Created Files |
|-------|-----------|---------------|
| `@` | `player` | `projects/{id}/pieces/player_{x}_{y}/` |
| `&` | `npc` | `projects/{id}/pieces/npc_{x}_{y}/` |
| `Z` | `zombie` | `projects/{id}/pieces/zombie_{x}_{y}/` |
| `T` | `chest` | `projects/{id}/pieces/chest_{x}_{y}/` |

**Generated Files:**
- `state.txt` - Position, type, behavior defaults
- `piece.pdl` - Piece DNA with methods

### 2.4 Backspace Destructive Editing

**NEW:**
```c
// Backspace - Destructive Editing
if (key == 127 || key == 8) {  // Backspace or Ctrl+H
    if (gui_focus_index == 1 && map_count > 0) {
        char cmd[MAX_PATH];
        snprintf(cmd, sizeof(cmd), "%s/pieces/apps/editor/ops/+x/place_tile.+x %s %d %d . > /dev/null 2>&1",
                 project_root, project_maps[map_idx], cursor_x, cursor_y);
        system(cmd);
        set_response("Tile cleared");
    }
}
```

**Behavior:**
- Reverts tile to default `.`
- Uses editor's place_tile (logs to ledger for undo)

## 6. OP INTERFACE SUMMARY

### 6.1 move_selector.+x
```bash
Usage: ./move_selector.+x <direction> [project_id]
Directions: up, down, left, right
Example: ./move_selector.+x up

Updates:
- pieces/apps/editor/pieces/selector/state.txt (pos_x, pos_y)
- pieces/apps/editor/state.txt (cursor_x, cursor_y)

Signals:
- pieces/display/frame_changed.txt
- pieces/apps/editor/view_changed.txt
```

### 6.2 create_piece.+x
```bash
Usage: ./create_piece.+x <type> <x> <y> [project_id]
Types: @, &, Z, T, *, X or player, npc, zombie, chest, item, selector
Example: ./create_piece.+x npc 7 3

Creates:
- projects/{id}/pieces/{type}_{x}_{y}/state.txt
- projects/{id}/pieces/{type}_{x}_{y}/piece.pdl

Logs:
- projects/{id}/config/editor_history.ledger (CREATE_PIECE|type|x|y)
```

### 6.3 undo_action.+x
```bash
Usage: ./undo_action.+x [project_id]
Example: ./undo_action.+x demo_1

Reads:
- projects/{id}/config/editor_history.ledger (last entry)

Executes:
- Inverse operation (revert tile, delete piece)

Updates:
- Ledger (pops last line)
```
