## HOW TO TEST MANUALLY

```bash
cd /path/to/project

# 1. Clean start
./kill_all.sh
sleep 2

# 2. Set up man-ops state
echo "project_id=man-ops" > pieces/apps/player_app/manager/state.txt
echo "active_target_id=selector" >> pieces/apps/player_app/manager/state.txt

# 3. Reset selector
cat > projects/man-ops/pieces/selector/state.txt << 'EOF'
name=Selector
type=selector
pos_x=5
pos_y=5
pos_z=0
on_map=1
icon=X
EOF

# 4. Start system
./run_chtpm.sh &
sleep 3

# 5. Trigger layout
echo "pieces/apps/man-ops/layouts/man-ops.chtpm" > pieces/display/layout_changed.txt
sleep 2

# 6. Check module is running
ps aux | grep man-ops | grep -v grep

# 7. Test key injection
echo "1002" >> pieces/apps/player_app/history.txt  # UP key
sleep 0.5

# 8. Check state changed
cat projects/man-ops/pieces/selector/state.txt | grep pos

# 9. Render and view
./pieces/apps/playrm/ops/+x/render_map.+x
cat pieces/apps/player_app/view.txt | head -12
```
