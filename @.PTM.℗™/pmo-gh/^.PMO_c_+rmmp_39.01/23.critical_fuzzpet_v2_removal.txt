### 1. Loader Scans Projects Directory Dynamically
**File:** `pieces/apps/playrm/loader/loader_module.c`

**Code (line ~60-95):**
```c
void scan_projects() {
    char *projects_dir_path = NULL;
    if (asprintf(&projects_dir_path, "%s/projects", project_root) != -1) {
        DIR *dir = opendir(projects_dir_path);
        if (dir) {
            struct dirent *entry;
            while ((entry = readdir(dir)) != NULL && project_count < MAX_ITEMS) {
                if (entry->d_name[0] == '.') continue;
                // Adds ALL directories to project list
                strncpy(projects[project_count++], entry->d_name, MAX_LINE - 1);
            }
            closedir(dir);
        }
    }
    // ... generates loader.pdl with METHOD entries
}
```

**Conclusion:** The loader reads from `projects/` directory. If `fuzzpet_v2` shows, the directory MUST exist somewhere.
