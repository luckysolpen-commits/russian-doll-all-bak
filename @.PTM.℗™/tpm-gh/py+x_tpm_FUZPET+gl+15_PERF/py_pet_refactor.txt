# Python FuzzPet Refactor Guide

## Goal
Update the Python TPM FuzzPet implementation to match the enhanced C version with:
1. **Reliable frame change detection** using marker file size (not mtime)
2. **GL graphical renderer** alongside ASCII terminal renderer
3. **60 FPS update rate** for both renderers
4. **Dual input sources** (terminal + GL window keyboard)

---

## Current Architecture (Python)

```
run_fuzzpet.py → orchestrator.py
                    ├── keyboard.py (terminal input)
                    ├── game_logic.py
                    ├── renderer.py (ASCII only)
                    ├── response_handler.py
                    └── master_ledger_monitor.py
```

**File Flow:**
- `keyboard.py` → writes to `pieces/keyboard/history.txt`
- `game_logic.py` → reads history, writes to `pieces/display/current_frame.txt`
- `renderer.py` → reads `current_frame.txt` using mtime detection

---

## Target Architecture (After Refactor)

```
run_fuzzpet.py → orchestrator.py
                    ├── keyboard.py (terminal input)
                    ├── game_logic.py (writes frame + marker)
                    ├── renderer.py (ASCII - marker size detection)
                    ├── gl_renderer.py (NEW - GL window + keyboard + marker size detection)
                    ├── response_handler.py
                    └── master_ledger_monitor.py
```

**New File Flow:**
- `keyboard.py` → writes to `pieces/keyboard/history.txt`
- `gl_renderer.py` → ALSO writes to `pieces/keyboard/history.txt` (GL window input)
- `game_logic.py` → writes to `pieces/display/current_frame.txt` + appends to `pieces/master_ledger/frame_changed.txt`
- `renderer.py` → checks `frame_changed.txt` SIZE for updates
- `gl_renderer.py` → checks `frame_changed.txt` SIZE for updates

### New Implementation (FIXED)
```python
#!/usr/bin/env python3
"""
Renderer plugin - monitors frame_changed.txt marker file for reliable updates
Based on proven C renderer.c implementation
"""

import os
import sys
import time
from datetime import datetime

script_dir = os.path.dirname(__file__)
CURRENT_FRAME_PATH = os.path.join(script_dir, '../pieces/display/current_frame.txt')
MARKER_FILE_PATH = os.path.join(script_dir, '../pieces/master_ledger/frame_changed.txt')

# Track marker file SIZE (not mtime!)
last_marker_size = 0


def render_display():
    """Render the display content from current_frame.txt"""
    if os.path.exists(CURRENT_FRAME_PATH):
        try:
            with open(CURRENT_FRAME_PATH, 'r', encoding='utf-8') as frame_file:
                content = frame_file.read()
                print(content, end='')
                sys.stdout.flush()  # Force flush for immediate display
        except IOError:
            show_default_display()
    else:
        show_default_display()


def show_default_display():
    """Show default display content"""
    default = """==================================
       FUZZPET DASHBOARD        
==================================
Pet Name: Fuzzball
Hunger: 50
Happiness: 75
Energy: 100
Level: 1
Position: (5, 5)

GAME MAP:
####################
#  ...............T#
#  ...............T#
#  ....R@.........T#
#  ....R..........T#
#  ....R..........T#
#  ....R..........T#
#  ................#
#                  #
####################

Controls: w=up, s=down, a=left, d=right | 1=feed, 2=play, 3=sleep, 4=status, 5=evolve, 6=end_turn
==================================
"""
    print(default)


def main():
    """Main loop - check marker file SIZE every 16.667ms (60 FPS)"""
    global last_marker_size
    
    print("Renderer Service Started")
    print("Monitoring pieces/master_ledger/frame_changed.txt for changes...")
    print("Running at 60 FPS")
    print("Press Ctrl+C to stop renderer.\n")
    
    # Get initial marker file size
    if os.path.exists(MARKER_FILE_PATH):
        last_marker_size = os.path.getsize(MARKER_FILE_PATH)
    
    # Main monitoring loop - 60 FPS
    while True:
        try:
            # Check marker file SIZE (reliable, no mtime issues!)
            if os.path.exists(MARKER_FILE_PATH):
                current_size = os.path.getsize(MARKER_FILE_PATH)
                
                if current_size != last_marker_size:
                    # Frame was updated!
                    render_display()
                    last_marker_size = current_size
            
            # Sleep 16.667ms for 60 FPS
            time.sleep(0.016667)
            
        except KeyboardInterrupt:
            print("\nRenderer stopped.")
            break
        except Exception as e:
            print(f"Renderer error: {e}", file=sys.stderr)
            time.sleep(0.1)


if __name__ == "__main__":
    main()
```

### Key Changes
| Aspect | Old | New |
|--------|-----|-----|
| Detection | mtime (1-second resolution) | Marker file SIZE (exact) |
| Update rate | 100ms (10 FPS) | 16.667ms (60 FPS) |
| File checked | `current_frame.txt` mtime | `frame_changed.txt` size |
| Flush | None | `sys.stdout.flush()` |

## Part 2: Add GL Renderer (gl_renderer.py - NEW FILE)

Create `plugins/gl_renderer.py`:

```python
#!/usr/bin/env python3
"""
GL Renderer - OpenGL window display with keyboard input
Reads from pieces/display/current_frame.txt (same as ASCII renderer)
Writes to pieces/keyboard/history.txt (same as keyboard.py)
Checks pieces/master_ledger/frame_changed.txt for updates
"""

import os
import sys
import time
from datetime import datetime

# PyOpenGL imports
try:
    from OpenGL.GL import *
    from OpenGL.GLU import *
    from OpenGL.GLUT import *
except ImportError:
    print("PyOpenGL not installed. Install with: pip install PyOpenGL PyOpenGL_accelerate", file=sys.stderr)
    sys.exit(1)

# Constants
WINDOW_WIDTH = 800
WINDOW_HEIGHT = 1200
BACKGROUND_COLOR = (0.1, 0.1, 0.1, 1.0)

# File paths
script_dir = os.path.dirname(__file__)
CURRENT_FRAME_PATH = os.path.join(script_dir, '../pieces/display/current_frame.txt')
MARKER_FILE_PATH = os.path.join(script_dir, '../pieces/master_ledger/frame_changed.txt')
KEYBOARD_HISTORY_PATH = os.path.join(script_dir, '../pieces/keyboard/history.txt')
MASTER_LEDGER_PATH = os.path.join(script_dir, '../pieces/master_ledger/master_ledger.txt')

# Global state
frame_content = ""
last_marker_size = 0

# Keyboard constants (match C version)
CTRL_KEY = lambda k: k & 0x1f
ARROW_UP = 1002
ARROW_DOWN = 1003
ARROW_LEFT = 1000
ARROW_RIGHT = 1001


def write_keyboard_input(key_code):
    """Write key press to keyboard history (same format as keyboard.py)"""
    try:
        os.makedirs(os.path.dirname(KEYBOARD_HISTORY_PATH), exist_ok=True)
        
        timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        
        # Write to keyboard history
        with open(KEYBOARD_HISTORY_PATH, 'a') as fp:
            fp.write(f"[{timestamp}] KEY_PRESSED: {key_code}\n")
        
        # Log to master ledger
        with open(MASTER_LEDGER_PATH, 'a') as ledger:
            ledger.write(f"[{timestamp}] InputReceived: key_code={key_code} | Source: gl_keyboard_input\n")
            
    except Exception as e:
        print(f"Error writing keyboard input: {e}", file=sys.stderr)


def on_keyboard(key, x, y):
    """GLUT keyboard callback"""
    key_code = ord(key) if isinstance(key, str) else key
    
    # Handle Ctrl+C
    if key_code == CTRL_KEY(ord('c')):
        glutLeaveMainLoop()
        return
    
    write_keyboard_input(key_code)


def on_special_keyboard(key, x, y):
    """GLUT special key callback (arrow keys, etc.)"""
    key_code = 0
    
    if key == GLUT_KEY_UP:
        key_code = ARROW_UP
    elif key == GLUT_KEY_DOWN:
        key_code = ARROW_DOWN
    elif key == GLUT_KEY_LEFT:
        key_code = ARROW_LEFT
    elif key == GLUT_KEY_RIGHT:
        key_code = ARROW_RIGHT
    else:
        return
    
    write_keyboard_input(key_code)


def parse_frame_file():
    """Read frame content from current_frame.txt"""
    global frame_content
    
    try:
        if os.path.exists(CURRENT_FRAME_PATH):
            with open(CURRENT_FRAME_PATH, 'r', encoding='utf-8') as fp:
                frame_content = fp.read()
    except Exception as e:
        print(f"Error reading frame file: {e}", file=sys.stderr)


def on_display():
    """GLUT display callback"""
    glClear(GL_COLOR_BUFFER_BIT)
    
    glMatrixMode(GL_PROJECTION)
    glLoadIdentity()
    glOrtho(0, WINDOW_WIDTH, 0, WINDOW_HEIGHT, -1, 1)
    glMatrixMode(GL_MODELVIEW)
    glLoadIdentity()
    
    # Render frame content as text using GLUT bitmap font
    if frame_content:
        line_height = 20.0
        start_x = 50.0
        start_y = WINDOW_HEIGHT - 50.0
        
        lines = frame_content.split('\n')
        for i, line in enumerate(lines):
            y = start_y - (i * line_height)
            glRasterPos2f(start_x, y)
            for char in line:
                glutBitmapCharacter(GLUT_BITMAP_HELVETICA_18, ord(char))
    
    glutSwapBuffers()


def on_reshape(w, h):
    """GLUT reshape callback"""
    glViewport(0, 0, w, h)


def on_timer(value):
    """GLUT timer callback - check for frame updates"""
    global last_marker_size, frame_content
    
    # Check marker file SIZE
    if os.path.exists(MARKER_FILE_PATH):
        current_size = os.path.getsize(MARKER_FILE_PATH)
        
        if current_size != last_marker_size:
            # Frame was updated!
            parse_frame_file()
            last_marker_size = current_size
    
    # Always redraw
    glutPostRedisplay()
    
    # Schedule next timer (16ms = ~60 FPS)
    glutTimerFunc(16, on_timer, 0)


def init_gl():
    """Initialize OpenGL"""
    glClearColor(*BACKGROUND_COLOR)


def main():
    """Main entry point"""
    global last_marker_size
    
    print("GL Renderer Service Started")
    print(f"Window size: {WINDOW_WIDTH}x{WINDOW_HEIGHT}")
    print("Reading from: pieces/display/current_frame.txt")
    print("Writing keyboard input to: pieces/keyboard/history.txt")
    print("Checking marker: pieces/master_ledger/frame_changed.txt")
    print("Press Ctrl+C in GL window to stop.\n")
    
    # Initialize GLUT
    glutInit(sys.argv)
    glutInitDisplayMode(GLUT_DOUBLE | GLUT_RGB)
    glutInitWindowSize(WINDOW_WIDTH, WINDOW_HEIGHT)
    glutCreateWindow(b"TPM FuzzPet - GL Renderer")
    
    # Set callbacks
    glutDisplayFunc(on_display)
    glutReshapeFunc(on_reshape)
    glutKeyboardFunc(on_keyboard)
    glutSpecialFunc(on_special_keyboard)
    
    # Initialize
    init_gl()
    parse_frame_file()
    
    # Get initial marker size
    if os.path.exists(MARKER_FILE_PATH):
        last_marker_size = os.path.getsize(MARKER_FILE_PATH)
    
    # Start timer (16ms = ~60 FPS)
    glutTimerFunc(16, on_timer, 0)
    
    print("GL window opened. Type in the GL window to control the pet.")
    print("Both GL window and terminal input work simultaneously!\n")
    
    # Enter main loop
    glutMainLoop()
    
    print("GL Renderer stopped.")


if __name__ == "__main__":
    main()
```

## Troubleshooting

### GL window doesn't open
- Check PyOpenGL is installed: `pip install PyOpenGL PyOpenGL_accelerate`
- Check you have a display server running (X11, Wayland)

### Updates are slow/stuttering
- Verify marker file is being written: `cat pieces/master_ledger/frame_changed.txt`
- Check file size grows with each update: `ls -la pieces/master_ledger/frame_changed.txt`

### Only one renderer works
- Check orchestrator launches both: look for both PIDs in startup output
- Check both are reading same marker file

### Keyboard input not working from GL window
- Check GL window has focus
- Verify `gl_renderer.py` is writing to correct history file
- Check file format matches: `[TIMESTAMP] KEY_PRESSED: CODE`
