Temporal Element Guide


How to use this Chart:

Name: element's name
LV: element's base level
+/-: allocation range of the element
Description: description of the element
Type: magic category the element belongs to
Range: who the element affects
$: cost of the element (T-element must be trapped, E-element is attained by defeating an enemy)
Location: where the element can be found or purchased (A-Arni, T-Termina, Gh-Guldove Home, Ga-Guldove Another, Mh-Marbule Home, Ma-Marbule Another), or the enemy which must be defeated/trapped
*Note that these are just examples: some elements can be found in multiple or many locations*
Notes: other details (C-Consumable, I-May only be used by a character of the same innate)

In the Temporal Element system, Elements are the core of combat in the Land of Time, replacing standard magic, items, and specialized techniques. They are categorized by six colors, each with specific attributes and opposing counterparts. 
## The Six Element Colors
Every character and enemy has an Innate Color, which determines their elemental strengths and weaknesses.

| Color | Associated Attributes | Weakness |
|---|---|---|
| Red | Fire and Magma | Blue |
| Blue | Water and Ice | Red |
| Green | Nature and Air | Yellow |
| Yellow | Earth and Electricity | Green |
| White | Light, Holy, and Life | Black |
| Black | Shadow, Space, and Gravity | White |

## Key Mechanics

* Element Grid: Before battle, you "allocate" Elements into a grid. Each slot has a level (1–8); placing an Element in a higher level makes it stronger, while a lower level weakens it.
* Charging in Battle: You cannot use Elements immediately. You must land physical attacks to build up "Element Levels" (1, 2, or 3 points depending on the attack's strength).
* Field Effect: Located in the top-left corner, this tracks the last three Elements cast. If the field is one solid color, Elements of that color deal massive damage, while opposing colors are weakened.
* Summons: These are the most powerful Elements and can only be cast when the entire Field Effect matches the Summon's color.
* Consumables & Techs: Some Elements are consumable items (like Tablets), while others are Character Techs—unique skills that only a specific character can use and are automatically assigned to their grid.
* Traps: Special Elements that can "catch" powerful spells from enemies. To use them, you must cast the trap before the enemy uses the specific spell.
