## Phase 0: Pre-Launch (Months 1-6)
### "Build the Community Before the Building"

**Goal:** Validate demand. Build audience. Create software. Zero physical risk.

### Actions:

#### Software (Priority #1)
- [ ] Build **TPMOJIO Digital** — web-based session player with multi-voice audio
- [ ] Build **Quest Board App** — digital quest board for communities
- [ ] Build **Session Recording Tools** — for DMs to record and share sessions
- [ ] Open source everything. MIT license.
- [ ] Price: Free tier + $5-15/mo premium features

#### Community Building
- [ ] Release all TPMOJIO audio sessions publicly (YouTube, podcast platforms)
- [ ] Start Discord server for TPMOJIO fans
- [ ] Run **pop-up events** at libraries, colleges, game stores
- [ ] Host virtual TPMOJIO sessions (free, open to anyone)
- [ ] Build email list. Target: 1,000 subscribers by Month 6.

#### Content
- [ ] Continue producing audio sessions (Clone Chronicles, HARD_VVAR, etc.)
- [ ] Start YouTube channel with session highlights, character art, lore videos
- [ ] Write TPMOJIO Cafe lore (the in-universe origin story)
- [ ] Create social media presence (TikTok, X, Instagram)

#### Validation Metrics
- [ ] 500+ Discord members
- [ ] 1,000+ email subscribers
- [ ] 100+ active software users (free or paid)
- [ ] 3+ successful pop-up events (20+ attendees each)
- [ ] 50+ people who say "I'd pay for a membership"

#### Budget: $5,000-15,000
- Domain, hosting, software tools: $2,000
- Pop-up event costs: $3,000
- Marketing (social ads, merch for events): $3,000
- Contingency: $2,000-7,000

**Revenue target:** $500-2,000/mo from software by Month 6.

## The "What If It Fails" Plan

### If Phase 0 fails (no community, no software users):
- **Loss:** $5,000-15,000
- **Lesson:** The concept isn't resonating. Pivot to something else.
- **Silver lining:** You still have the IP, the audio content, the open source software.

### If Phase 1 fails (store doesn't hit $60K/mo):
- **Loss:** $150,000-280,000
- **Lesson:** The physical model doesn't work in this market.
- **Pivot:** Focus on software. Sell Quest Board App to other stores. License TPMOJIO IP.

### If Phase 2 fails (second location underperforms):
- **Loss:** $200,000-250,000
- **Lesson:** The model isn't replicable.
- **Pivot:** Keep Location 1. Focus on making it the best it can be. Grow software.

### If Phase 3 fails (regional expansion stalls):
- **Loss:** $1-2M
- **Lesson:** You scaled too fast.
- **Pivot:** Close underperforming locations. Focus on profitable ones. Grow software.

### If Phase 4 fails (national expansion fails):
- **Loss:** $10-20M
- **Lesson:** You're not a national brand.
- **Pivot:** Regional brand is still valuable. Sell to larger company or stay regional.
