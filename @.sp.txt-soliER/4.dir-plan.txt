## Book 4: RedWhiteAndU (CYOA) — Horli Voice — SPECIAL HANDLING

**Path:** `SP.all-writeez28-a1/#.1-13.sp.redwhiteandU]🇺🇸️🇬🇱️b2/`

### This is NOT a normal book. It's CYOA (Choose Your Own Adventure).

**Structure:**
| Book | Path | Pages | Chars | Notes |
|------|------|-------|-------|-------|
| **Book 0** | `00_book]kOT]a0]gmii/rb&u.CYOA_Book.0/` | 55 | ~29K | Clone graduation |
| **Book 1** | `01]ugem&]gmii/rb&u.CYOA_Book]01/` | 100 | ~? | Continuation |
| **Book 3** | `3&4.cd&cclonesdrones&crx/3.rb&u.CYO_book.3]CD&C.X/` | 297 | ~536K | Massive branching |
| **Book 4** | `3&4.cd&cclonesdrones&crx/4.rb&u.cdc.2.book.4]BEST/rb&u.cyoa_book4]cd&c2/` | 150 | ~164K | Best version |

### CYOA Format:
- Second-person narrative: "You are Clone 734..."
- Pages contain **choices** that branch to other pages
- Pages reference each other by number (e.g., "Go to page 42")

### CYOA Audio Strategy — "Choose Audio File":
Instead of one linear audio file, create **one MP3 per page**:
- `rb&u.book0.page01.mp3`, `rb&u.book0.page02.mp3`, etc.
- Each page's audio is independent
- Player listens to page audio, then manually selects next page's audio file
- Can be assembled into a simple HTML/CLI player later with clickable choices

### Script: `batch_convert_cyoa_edge_tts.py`
- Iterates all `page_NN.txt` files in a CYOA book dir
- Converts each page independently (no aggregation)
- Output: `rb&u.book{N}.page{NN}-v1.0.mp3`
- Run once per book (0, 1, 3, 4)

### Notes:
- Book 3 has pages going up to `page_1001.txt` — massive
- Page files use zero-padded naming: `page_01.txt`, `page_100.txt`
- Some pages may be very short (choice nodes) or very long (story segments)
- Has `outline.txt` and `navigation_guide.txt` files for reference
