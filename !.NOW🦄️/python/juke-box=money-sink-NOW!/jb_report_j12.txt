Jukebox Player - January 12th Report

LIMITATIONS DUE TO PYGAME.MIXER:

1. SEEKING FUNCTIONALITY:
   - Current UI includes a seek slider but pygame.mixer does not support seeking to 
     arbitrary positions in MP3 files
   - User can drag the progress bar/slider but actual audio seeking is not possible
   - To implement true seeking, would need to switch to audio libraries like:
     * pydub with pyaudio
     * python-sounddevice
     * pygame.sndarray with manual seeking

2. PITCH SHIFTING:
   - Pitch controls are implemented in the UI (EQ tab) but pygame.mixer does not 
     support real-time pitch shifting
   - Setting is stored but has no audible effect due to library limitations
   - To implement true pitch shifting, would need advanced audio libraries like:
     * pydub with soundstretch
     * librosa 
     * rubberband-python
