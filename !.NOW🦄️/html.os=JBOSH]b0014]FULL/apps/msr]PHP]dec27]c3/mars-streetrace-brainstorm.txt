## TECHNICAL ARCHITECTURE PLAN

### File Structure
```
mars-streetrace-raider/
├── index.html          (main game interface)
├── css/
│   └── terminal.css    (cyberpunk terminal styling)
├── js/
│   ├── game.js         (main game loop)
│   ├── market.js       (stock/commodity trading)
│   ├── player.js       (player data management)
│   ├── corporations.js (corp generation/management)
│   ├── save.js         (CSV save/load system)
│   └── clock.js        (time progression)
├── php/
│   ├── save_game.php   (write CSV to server)
│   ├── load_game.php   (read CSV from server)
│   └── clear_temp.php  (delete old saves)
└── data/
    └── temp/           (temporary CSV storage)
```

### CSV Data Structure

**players.csv:**
```csv
name,cash,type,ticker_on
PlayerName,10000.00,HUMAN,false
```

**corporations.csv:**
```csv
ticker,name,industry,price,shares_outstanding,free_cash,total_assets,net_worth
RPL,Redplanet Labs,BIOTECH,50.25,100.0,500.0,2000.0,1500.0
CMF,Clown Mafia Front,ENTERTAINMENT,12.50,50.0,100.0,500.0,300.0
```

**portfolio.csv:**
```csv
player,ticker,shares,avg_price
PlayerName,RPL,10,48.50
PlayerName,CMF,25,11.75
```

**game_state.csv:**
```csv
key,value
current_year,2025
current_month,1
current_day,1
current_turn,1
interest_rate,2.5
```

### PHP Save/Load System

**save_game.php** - receives JSON, converts to CSV, stores on server
**load_game.php** - reads CSV, converts to JSON, sends to client
**download_save.php** - packages all CSVs into zip for user download
**upload_save.php** - accepts zip upload, extracts CSVs to server
